# Border & Shadows Showcase

A React application built with Vite and Tailwind CSS to demonstrate advanced border and shadow effects.

## Live Demo
[https://krugou.github.io/bordershadows/](https://krugou.github.io/bordershadows/)

## Tech Stack
- React
- Vite
- Tailwind CSS
- TypeScript

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
